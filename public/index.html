Project Overview

A modern, scalable chat application with real-time messaging and user management capabilities.

Key Features

User Authentication

Secure user registration and login with JWT token-based authentication.

Real-Time Messaging

Instant message sending and receiving across multiple conversations.

Conversation Management

Create, join, and manage multiple conversation rooms with ease.

User Discovery

Retrieve and interact with a list of all registered users.

Conversation History

Access and retrieve message history for each conversation.

Dynamic User Addition

Add new users to existing conversations seamlessly.

Authentication Routes (/api/auth)

POST /signup

Register a new user with email, password, and profile details

POST /login

Authenticate user and generate JWT token

Chat Routes (/api/chat)

POST /conversations

Create a new conversation/chat room

POST /messages

Send a new message in a conversation

GET /conversations/:conversationId/messages

Retrieve all messages for a specific conversation

GET /users/:userId/conversations

Fetch all conversations for a specific user

POST /conversations/add-user

Add a new user to an existing conversation

GET /users

Retrieve list of all registered users
